# Webmusic

简体中文：

一个网页播放器，将xxx.lrc（歌词）、xxx.png（歌曲图片）、xxx.mp3放置到xxx文件夹。请注意：文件夹名称和其他文件的名称必须一样，例如：播放Unstoppable，文件夹名称：Unstoppable，文件夹内文件：Unstoppable.png、Unstoppable.lrc、Unstoppable.mp3。网址：https://kevinananda2026.github.io/Webmusic

English:

A web player: place xxx.lrc (lyrics), xxx.png (song image), and xxx.mp3 into the xxx folder. Please note: the folder name and the names of the other files must be the same. For example, to play "Unstoppable", the folder name should be "Unstoppable", and the files inside the folder should be: Unstoppable.png, Unstoppable.lrc, Unstoppable.mp3.URL：https://kevinananda2026.github.io
